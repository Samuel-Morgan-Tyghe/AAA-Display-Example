# AAA installation

To start the Project run run these two commands

#### json-server --watch AAAdb.json

#### npm start

## What can be improved

- encrypt passwords


## Troubleshoot
If "'react-scripts' is not recognized as an internal or external command"
run: 
#### npm i react-scripts
